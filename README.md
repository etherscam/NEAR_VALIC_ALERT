# NEAR_VALIC_ALERT
Script for alert (call and message telegram) for NEAR Validators  
 ### For deployment:

Clone repo
```
sudo apt-get update
sudo apt-get install git
git clone https://github.com/etherscam/INCH.git
```
Fill in the file near_alarm.ini
```
nano near_alarm.ini
```
Star script
```
Python3 ALERT.py
```
